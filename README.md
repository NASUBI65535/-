動画圧縮ツール


製作者　NASUBI



動画圧縮ツール.exeをダブルクリックして起動してください


ドラッグ＆ドラッグでファイルパスを入れられます


ディレクトリを指定するとそのディレクトリにある指定した拡張子の動画ファイルを全て圧縮し、

resultフォルダにまとめて出力します
